# 3D Market Radar

3D Market Radar is an AI-powered system that analyzes 3D printable models and evaluates their commercial potential in marketplaces like Etsy.

## Features

* Scrapes newly uploaded 3D models from platforms like MakerWorld
* Uses AI to evaluate product-market fit
* Analyzes Etsy marketplace demand
* Estimates potential revenue
* Identifies high-opportunity products automatically

## Use Case

This project helps makers and developers identify which 3D models are worth printing and selling.

## Status

This is a non-commercial experimental project built for learning and research purposes.
